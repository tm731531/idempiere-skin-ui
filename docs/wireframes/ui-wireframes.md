# UI Wireframes 畫面草圖

> 專案：iDempiere 醫療診所 UI
> 版本：1.0
> 日期：2026-02-06

---

## 設計原則

1. **Mobile First** - 以手機尺寸為主要設計
2. **大按鈕** - 方便觸控操作
3. **一頁一事** - 每個畫面專注一件事
4. **即時反饋** - 操作後立即顯示結果
